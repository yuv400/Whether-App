🌦️ Weather and Map App
A simple Weather and Map application built using HTML, CSS, and JavaScript. The app fetches real-time weather information for any city and displays its location on an interactive map.

🚀 Live Demo
https://whether-app-nu-two.vercel.app/

Features
Search weather by city name
Display real-time weather information
Show minimum and maximum temperature
Display wind speed and direction
Show cloud coverage percentage
Display sunrise and sunset timings
Interactive city map integration
Clean and responsive user interface

Technologies Used
HTML
CSS
JavaScript
OpenWeather API
Google Maps API

How It Works
Enter a city name in the search box.
Click the Search button.
The app fetches weather data from the API.
Weather details are displayed instantly.
The city location is shown on the map.

Learning Outcomes
API Integration
Fetch API & Async JavaScript
DOM Manipulation
Event Handling
Working with Maps APIs
